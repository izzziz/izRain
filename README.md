# izRain

IzRain is an easy to use weather application that incorporates cocoa pods, the core location frameworks and JSON parsing to fetch live weather data from OpenWeatherMap. This app built with Appleâ€™s SF Symbols and Dark Mode enabled available iOS 13.

## Tech
- Working with openWeather API by making HHTP requests with URLSession
- Parse JSON with the native JSONDecoder
- Using computed properties, closures and extensions
- Using Core location to get the GPS data
